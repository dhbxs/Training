# Training
一个简陋的小型购物系统
---
## 项目由来
---
该项目是学校Jsp课程设计专周作业，实现注册，登陆，加购物车，下单流程，以及后台添加删除商品功能。
---
## 项目用到的基础知识及工具

1. 基础Java知识
2. JavaWeb Servlet，jsp，MVC(Model-View-Controller)设计模式
3. JDBCUtils
4. DBUtils
5. JUnit4
6. Tomcat9.0.35
7. BootStrap框架
---

## 项目功能
### 前台功能
1. 购物网站的用户注册，登陆功能
2. 浏览商品，选择需要添加到购物车
3. 从购物车删除商品
4. 商品下单，完成购物
### 后台功能
1. 后台用户登陆
2. 管理员查看现有的商品，销量，库存等信息
3. 管理员删除商品
4. 管理员添加商品到库存
